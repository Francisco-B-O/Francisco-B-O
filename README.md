# 💫 About Me
👨‍💻 I am a Software Engineer and Full Stack Developer with professional experience building enterprise applications and software solutions. My profile combines backend and frontend development, allowing me to participate in all stages of the software lifecycle, from analysis and design to implementation, deployment and maintenance.

💼 Throughout my professional career, I have worked on the development, maintenance and evolution of enterprise applications, building scalable and maintainable solutions using technologies such as Java, Spring Boot and Angular. I have experience designing and integrating REST APIs, working with microservices and microfrontend architectures, database management, and applying software engineering best practices in real-world projects.

⚙️ I have also participated in CI/CD processes, deployment automation and software quality initiatives using tools such as GitHub Actions, GitLab, SonarQube and Fortify. In addition, I have taken part in technical coordination activities, supporting development teams, tracking project progress and contributing to technical decision-making.

🤖 Beyond software engineering, I am strongly focused on Artificial Intelligence and Big Data. I work with technologies and tools oriented towards large-scale data processing, machine learning and intelligent systems, combining software development with data-driven solutions.

📊 My experience includes working with technologies such as Python, Apache Spark, Databricks, Pandas, NumPy, Scikit-Learn and PyTorch, participating in data preparation, transformation, exploratory analysis, model training and evaluation, as well as the development of solutions focused on extracting value from data.

🚀 Currently, I am investing a significant part of my learning journey in AI Engineering, Machine Learning, Deep Learning, Data Engineering, Big Data Architectures, Distributed Systems and MLOps, continuously exploring new technologies and methodologies that bridge the gap between software engineering and artificial intelligence.

🌱 Passionate about technology, software architecture, scalable systems and continuous learning. I enjoy working on challenging projects where software, data and AI come together to create impactful solutions.
